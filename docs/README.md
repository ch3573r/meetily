## Cleanup Policy

Keep this folder product-facing and current. Historical notes should either be
updated into current implementation docs or moved out of the public docs path.

Do not commit screenshots or examples that reveal private meetings, attendee
names, tenant URLs, tokens, local machine paths, or logs.
